(((가나다:::		→ 작은 글자
{{{가나다:::		→ 중간 글자(기본)
[[[가나다:::		→ 큰 글자
>>>가나다:::	→ 기본 강조색(초록)
>>r>>가나다:::	→ 강조색(빨강)
>>g>>가나다:::	→ 강조색(초록)
>>b>>가나다:::	→ 강조색(파랑)
>>y>>가나다:::	→ 강조색(노랑)
>>p>>가나다:::	→ 강조색(보라)
